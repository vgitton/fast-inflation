How to read this documentation?

This documentation is organized in modules. A module is a group of classes and functions together with explanative text that, conceptually, aim at achieving a specific goal. Thus, it is recommended to browse the modules page, and depending on one's interest, to read up about the different modules.

Recommended starting point

The simplest application to start from is probably user::srb_vis_222_weak, see user::srb_vis_222_weak::run(). This is also the fastest application to run because of the low numbers of variables (associated to inflation events) and constraints. To reproduce the proof of nonlocality of the Elegant Joint Measurement, please refer to the EJM module.
