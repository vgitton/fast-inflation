inf::Marginal::Evaluator Class Reference

#include "inf/constraints/marginal.h"

Class description

This class contains minimal data allowing to perform the operations described in inf::Marginal.

The idea is that the inf::Marginal::Evaluator contains "compiled" data from an inf::Marginal, allowing efficient extraction of marginal events from marginal permutations \(\redpermutedmargs\infmarg\) and an inflation event \(\infevent\in\infevents\). Furthermore, this is a minimal data structure that will be duplicated to be held by different threads in multithreading applications. Internally, it stores a representation of the set

\[ \{\extract{\infevent}{\margperm}\}_{\margperm\in\redpermutedmargs\infmarg}, \]

and allows the user to update individual outcomes of the inflation event \(\infevent\), which updates the above set accordingly. This allows the tree-based evaluation of inner products (see inf::TreeOpt) to perform the minimum amount of work at terminal nodes.

◆ set_scale_reference()

void inf::Marginal::Evaluator::set_scale_reference ( Num const *  scale)

This sets the reference scale \(\consscale\) that will be multiplied to the result of the evaluation described in inf::Marginal.

This scale depends in practice on the target distribution \(\targetp\), which means that it will typically change during nonlocality tests of successive distributions, as e.g. done in inf::VisProblem. Furthermore, this scale depends on the overall constraint set \(\constraintlist \subset \infconstraints\), which is why this is computed by inf::ConstraintSet and then passed here as a reference.
