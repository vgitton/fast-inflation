◆ test_dual_vector_eval()

void user::test_dual_vector_eval ( inf::TargetDistr::ConstPtr const &  d)

This function tests the evaluation of inner products, see inf::DualVector and inf::Marginal.

It evaluates the inner product between a random inf::DualVector and an inflation marginal, where the inflation size is also random. The evaluation is compared using different methods for consistency checks.

Parameters
dThis distribution is used to define an inflation problem for which inner products will be evaluated.
