Module description

To specify and manipulate inflation constraints of the form \(\infq_{\infmarg_0\cdots\infmarg_{k-1}\infmargg} = \targetp_{\infmarg_0}\cdots\targetp_{\infmarg_{k-1}}\cdot\infq_{\infmargg}\).

The top-level class is inf::ConstraintSet, which handles a set of inf::Constraint. The user inputs constraints in the format described in inf::ConstraintParser, which is the class in charge of interpreting the user's text description of the constraints. The class inf::DualVector represents a dual vector acting on a constraint, and the class inf::Marginal is used to efficiently evaluate the inner product of an inf::DualVector with a deterministic inflation distribution.

◆ Quovec

typedef std::vector<Num> inf::Quovec

A "quotiented vector" denotes an element of some \(\quovecspace{\infmarg}\) for some \(\infmarg\subset\infparties\).

Given an inflation constraint \(\constraintname = (\infmarg_0,\dots,\infmarg_{k-1},\infmargg) \in \infconstraints\), the group of constraint symmetries \(\marggroup\) denotes all the inflation symmetries of \(\distrgroup\) (see inf::Inflation::get_inflation_symmetries()) that leave both \(\infmarg_0 \cup \dots \cup \infmarg_{k-1}\) and \(\infmargg\) invariant. Then, for an inflation marginal \(\infmarg \in \{\infmarg_0\cup\cdots\infmarg_{k-1}\cup\infmargg, \infmargg\}\), the reduced set of marginal events is \(\repr{\events\infmarg \setminus \marggroup}\), and the quovec vector space \(\quovecspace\infmarg\) is the vector space of all functions \(\repr{\events\infmarg\setminus\marggroup} \to \R\). (In fact, thanks to the way we approach exact arithmetic, a quovec has its image in \(\Z\).) A quovec \(\quovec\in\quovecspace\infmarg\) is occasionally stored in an inf::DualVector when we require the embedding \(\altquovecembed\infmarg(\quovec)\).
