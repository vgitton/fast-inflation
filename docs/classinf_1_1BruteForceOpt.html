Class description

This inf::Optimizer simply enumerate all inflation events to perform the minimization.

This class minimizes an inf::DualVector, representing some \(\quovec = \{\quovec_\constraintname\}_{\constraintname\in\constraintlist}\in\totquovecspace\), on the events of some inf::Inflation, i.e., solving the minimization problem

\[ \min_{\infevent\in\infevents} \inner{\quovec}{\totconstraintmap(\detdistr\infevent)}_{\constraintlist}, \]

where \(\constraintlist \subset \infconstraints\) describes the constraints of an inflation problem (see inf::ConstraintSet).

The minimization is simply performed by enumerating all \(\infevent\in\infevents\), in particular, ignoring the reduced set of events \(\redinfevents\) that uses symmetry.
