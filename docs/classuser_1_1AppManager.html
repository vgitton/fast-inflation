#include "user/app_manager.h"

Class description

This class parses the command-line arguments to try to run an application listed in user::get_application_list()

The main point of this class is to parse the command-line arguments, display adequate error messages and usage guides, and run applications when applicable. This enables the syntax ./{debug|release}_inf example_to_run [optional parameters], see the command-line parameters section.

See also
user::get_application_list() to register additional examples
