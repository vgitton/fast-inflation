Class description

This class represents an inflation marginal \(\infq_\infmarg\) for some set of inflation parties \(\infmarg \subset \infparties\) and allows to efficiently evaluate inner products.

To solve inflation problems with a Frank-Wolfe algorithm, and to verify nonlocality certificates, as shown in the paper, we require to evaluate the following expression:

\[ \consscale \cdot \sum_{\margperm \in \redpermutedmargs\infmarg} F(\extract{\infevent}{\margperm}), \]

where

  • \(\consscale \in \Z\) is a signed integer scale factor, incurred to be able to perform evaluation of inner products in integer arithmetic,
  • \(\infmarg \subset \infparties\) indicates an inflation marginal.
  • \(\infevent\in\infevents\) is an inflation event.
  • \(\redpermutedmargs\infmarg\) is the set of reduced marginal permutations associated to the symmetries of the inflation and the marginal \(\infmarg\). More specifically, we define \(\redpermutedmargs\infmarg = \repr{\permutedmargs\infmarg \setminus \marggroup}\), where \(\marggroup\) describes the group of constraint symmetries of some constraint \(\constraintname = (\infmarg_0,\dots,\infmarg_{k-1},\infmargg)\in \infconstraints\), and where \(\marggroup\) acts on \(\permutedmargs\infmarg\) with the action denoted \(\actt{\marggroup}{\permutedmargs\infmarg}\) in the paper. Recall that we have either \(\infmarg = \infmarg_0\cup\cdots\cup\infmarg_{k-1}\cup\infmargg\) or \(\infmarg = \infmargg\). The set \(\permutedmargs\infmarg\subset\margperms\infmarg\) describes the set of marginal permutations obtained as \(\act{\distrgroup}{\margperms\infmarg}(\gid, \id{\infmarg})\), where \(\distrgroup\) is described in inf::Inflation::UseDistrSymmetries, and where \((\gid,\id\infmarg) \in \margperms\infmarg\) describes the trivial marginal permutation.
  • \(\extract{\infevent}{\margperm} \in \events\infmarg\) is the marginal inflation event extracted from \(\infevent\) thanks to the marginal permutation \(\margperm\).
  • \(F \in \vecset{\infmarg}\) is a dual vector that is in practice of the form \(F = \altquovecembed\infmarg(\quovec)\) for some quotiented vector \(\quovec \in \quovecspace\infmarg\).

The class inf::Marginal allows to efficiently evaluate such expressions.

This class is tested in the application user::inflation_marginal.

Initialize with an inf::Inflation and a set of inflation party \(\infmarg\subset\infparties\) essentially.

The inf::EventTensor \(F\) relative to which the evaluation will take place will be specified in inf::Marginal::Evaluator.

Parameters
inflationThe inflation on which the inf::Marginal lives.
marginal_partiesThe set of inflation parties \(\infmarg\subset\infparties\) that describes the marginal we're referring to
constraint_groupThe group of constraint symmetries \(\marggroup\) obtained from an inf::Constraint. The relation between \(\infmarg\) and \(\constraintname = (\infmarg_{0},\dots,\infmarg_{k-1},\infmargg)\in\infconstraints\) is either \(\infmarg = \infmarg_{0}\cup\dots\cup\infmarg_{k-1}\cup\infmargg\) or \(\infmarg = \infmargg\), depending on whether we are dealing with the inflation marginal appearing in the left-hand side or the right-hand sideof the constraint \(\constraintname\).
store_boundsWe need to specify whether or not the relevant inf::EventTensor, storing the dual vector \(F\), stores upper or lower bounds allowing the branch-and-bound acceleration of the inflation event tree. Indeed, effectively, this changes the number of outcomes of the relevant inf::EventTensor, and this changes the representation of marginal events in inf::Marginal::Evaluator.
