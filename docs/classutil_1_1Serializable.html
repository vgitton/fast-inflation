Class description

Abstract class that defines the interface of a class that is serializable.

Note that we choose an implementation that does not require serializable objects to inherit from util::Serializable: as long as they implement a suitable method io(util::FileStream& stream), they're good to go.

virtual void util::Serializable::io ( util::FileStream &  stream)
pure virtual

This method defines what data is to be read/written to disk, and in what order.

Although the read/write mechanism is mostly meant to be implemented irrespectively of whether or not the object is being read/written, the object to be seralized can still implement sanity checks of the data being read by querying the util::FileStream::is_reading() method.

Implemented in inf::DualVector, inf::EventTree, and util::TestWritableClass.
