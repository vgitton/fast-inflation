Class description

template<typename T>
class util::ProductRange< T >

This class implements a range over std::vector<T>, akin to Python product ranges.

Equivalently, this class allows to iterate over the range of a Cartesian product of finite sets. Use as

for(std::vector<size_t> const& event : util::ProductRange(3,4))
{
// event = {0,0,0}
// event = {0,0,1}
// ...
// event = {3,3,3}
}

Product range with the same base for each entry of the list.

This allows to iterate over the Cartesian product \(\{0, \dots, \mathtt{base-1}\}^{\times \mathtt{n\_parties}} \). It is equivalent to util::ProductRange({base,...,base}).
