inf::EventTree::Node Class Reference

#include "inf/events/event_tree.h"

Class description

A node of the inf::EventTree, storing an outcome and some other inf::EventTreee::Node as children.

Note
This class is meant to be lightweight, which is why we want to avoid making it a virtual class. Hence, although an inf::EventTree::Node is effectively a util::Serializable, we do not make it explicitly so. (The reason is that a virtual class instance takes up a tiny bit more memory in C++.)
