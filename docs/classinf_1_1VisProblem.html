Class description

A simple dichotomic search for finding the minimum infeasible visibility of a distribution.

The class inf::VisProblem may be used for any family of distribution \(\{p_v\}_{v = v_\text{min}}^{v_\text{max}}\), indexed by an integer parameter \(v \in \mathcal V\). However, the dichotomic search that it performs only really make sense if the inflation relaxation will partition the set \(\mathcal{V}\) into \(\{v_\text{min},\cdots,v_0,v_1,\cdots,v_\text{max}\}\) such that

  • For all \(v \leq v_0\), \(p_v\) is compatible with the target inflation (for instance, \(p_v\) is local),
  • For all \(v \geq v_1\), \(p_v\) is incompatible with the target inflation (so that \(p_v\) is nonlocal).

In the paper, we describe a general class of noise models that generate families of distributions that satisfy these assumptions.

◆ get_minimum_nonlocal_visibility()

Num inf::VisProblem::get_minimum_nonlocal_visibility ( )

Get the minimum visibility \(v_1\) as described in inf::VisProblem.

Recall that \(v_1\) is the smallest value such that the associated inf::TargetDistr is provably nonlocal based on the specified inflation problem.

If even max_visibility is not ruled out, max_vibility + 1 is returned, which might be completely unphysical, but signals what happened.
