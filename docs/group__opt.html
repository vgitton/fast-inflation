Module description

This module describes how to efficiently minimize an inf::DualVector, representing some \(\quovec = \{\quovec_\constraintname\}_{\constraintname\in\constraintlist}\in\totquovecspace\), on the events of some inf::Inflation, i.e., solving the minimization problem.

\[ \min_{\infevent\in\infevents} \inner{\quovec}{\totconstraintmap(\detdistr\infevent)}_{\constraintlist}, \]

where \(\constraintlist \subset \infconstraints\) describes the constraints of an inflation problem (see inf::ConstraintSet). The class inf::Optimizer provides the general interface, and the two implementations that we propose are as follows:

  • inf::BruteForceOpt simply loops over all events \(\infevent \in \infevents\). This is mostly used to test that other means of optimizaitons give the expected results on relatively small instances.
  • inf::TreeOpt only looks at the symmetrized events \(\infevent \in \redinfevents\), represented as a tree with inf::Inflation::get_symtree(). It performs a tree search, and can be accelerated using a branch-and-bound approach, see inf::DualVector::StoreBounds.
