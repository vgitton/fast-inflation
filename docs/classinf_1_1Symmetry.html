Class description

A pair of party symmetry and outcome symmetry.

This class allows to encode the network and inflation symmetries. The inflation symmetries are described in the paper as the group \(\fullgroup\) with elements \(\fullsymexpl\) where \(\sourcesym \in \sourcegroup\), \(\partysym \in \partygroup\), \(\outsym \in \outgroup\). The inf::Symmetry class allows to store the inflation party permutation \(\act{\sourcesym}{\infparties} \circ \act{\partysym}{\infparties}\) together with the outcome permutation \(\outsym\).
