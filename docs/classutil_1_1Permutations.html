Class description

template<typename T>
class util::Permutations< T >

To iterate over permutations.

Example use:

for(std::vector<int> const& v : util::Permutations(3))
{
// v is successively
// {0,1,2}
// {0,2,1}
// {1,0,2}
// {1,2,0}
// {2,0,1}
// {2,1,0}
}
