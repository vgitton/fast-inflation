Class description

A distribution over an inf::Network.

In the notation of the paper, class instances represent a distribution \(\targetp \in \targetps\). This class allows the user to specify a target distribution over the triangle network, i.e., a distribution with support on a triangle network (represented as an inf::Network). It is then possible to test whether or not the distribution is causally compatible with the classical triangle network using an inf::FeasProblem instance.

A note on exact arithmetic

This class uses exact arithmetic for the probabilities. This requires the user to first choose a distribution with rational components, i.e., such that \(\targetp(\infevent) \in \mathbb{Q}\) for all \(\infevent\in\netevents\). and then to find an adequate common denominator for all probabilities. It is thus not possible to test exactly the nonlocality of a distribution that has irrational components with the present code. (In principle, this could be achieved, but would require fancier symbolic arithmetic capabilities.)

This functionality is tested in the application user::distribution_marginal.

Parameters
